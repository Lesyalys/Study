Установка:
1. Ставим 1С конфигурацию: Учебная
2. Создать информационную базу
3. На главной выбрать свою базу -> Конфигуратор
4. На верхней панели (где файлы, Правка, Конфигурация итд) выбрать Администрирование
5. Найти "Загрузить информационную базу"
6. Выбрать файл расширения dt  

_________________________________

1-4 --> Указывает на лабы, которые выполнены в БД, конкретно это значит в работе выполнены лабы с 1 по 4 включительно
